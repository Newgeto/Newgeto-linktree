# Newgeto — Linktree

Page de liens personnelle, sobre et animée : GitHub, Portfolio, LinkedIn et Discord.

Construite avec **React**, **Vite**, **Tailwind CSS** et plusieurs composants animés (Light Pillar, GradientText, ShinyText, SpotlightCard, ClickSpark).

## Développement

```bash
npm install
npm run dev
```

## Build de production

```bash
npm run build
npm run preview
```

## Déploiement

Hébergé sur **Vercel** (détection automatique du projet Vite). Le dossier de sortie est `dist/`.

## Personnalisation

Toutes les infos et les liens se modifient en haut de [`src/App.jsx`](src/App.jsx) (constantes `PROFILE` et `LINKS`).
